# 🔐 Password Manager Secure

Приложение для безопасного хранения паролей с графическим интерфейсом и надежным шифрованием.

## ✨ Особенности

- 🔒 **Шифрование Fernet (AES-128)** - надежная защита всех паролей
- 🔑 **Главный пароль** - защита с использованием PBKDF2 (100,000 итераций)
- 💾 **Локальное хранилище** - все данные остаются на вашем компьютере
- 🎨 **Графический интерфейс** - интуитивный и удобный (PyQt5)
- ➕ **Добавление паролей** - сохраняйте пароли для любых сайтов
- ✏️ **Редактирование** - обновляйте существующие пароли
- 🗑️ **Удаление** - удаляйте ненужные пароли
- 👁️ **Просмотр** - безопасный просмотр сохраненных паролей
- 📝 **Заметки** - добавляйте комментарии к каждому паролю
- ❓ **Встроенная справка** - подробная инструкция по использованию

## 🚀 Установка

### Требования:
- Python 3.7+
- pip

### Шаги установки:

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/qqe1843-collab/password-manager-secure.git
cd password-manager-secure
```

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Запустите приложение:**
```bash
python main.py
```

## 📦 Установка как EXE (Windows)

Если вы хотите распространять приложение без установки Python:

1. **Установите инструменты для сборки:**
```bash
pip install -r requirements-build.txt
```

2. **Запустите сборку:**
```bash
build.bat
```

3. **EXE файл будет создан в папке `dist/`**

Готовый файл `Password Manager Secure.exe` можно скачать и запустить на любом компьютере с Windows!

## 🔒 Безопасность

- ✓ Все пароли шифруются локально
- ✓ Главный пароль защищен PBKDF2
- ✓ Никакие данные не отправляются в интернет
- ✓ Полностью автономное приложение
- ✓ Открытый исходный код

## 📝 Использование

1. **Первый запуск:**
   - Создайте надежный главный пароль
   - Запомните его - восстановить невозможно!

2. **Добавление пароля:**
   - Нажмите кнопку "Добавить пароль"
   - Заполните данные сайта, логина и пароля
   - Добавьте заметку (необязательно)
   - Сохраните

3. **Управление паролями:**
   - Просмотр: выберите и нажмите "Показать пароль"
   - Редактирование: выберите и нажмите "Редактировать"
   - Удаление: выберите и нажмите "Удалить"

4. **Справка:**
   - Нажмите кнопку "Помощь" для подробных инструкций

## 💾 Где хранятся данные?

Все пароли сохраняются в файл `passwords.json` в папке приложения.

⚠️ **ВАЖНО:**
- Никогда не делитесь этим файлом
- Регулярно делайте резервные копии
- Храните копию в безопасном месте
